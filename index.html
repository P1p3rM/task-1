About Minecraft

Minecraft is an open world game where you run around creating anything you want and collecting food/resouces. To 'complete' the game you go to a realm called the end and fight the dragon. It is a cube based game so no circles exist inside the game. Minecraft is availabe on pretty much any device and is a fun game for all ages, with slight voilence but tollerable for younger ages.
